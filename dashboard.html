Review standard structural markers, display behaviors, and accessibility frameworks.

Block Elements

Block tags automatically claim a brand new line and expand horizontally to cover full available container width.

<div> block wrapper <p> paragraph tags
Inline Elements

Inline tags wrap around text. They render side-by-side and only consume horizontal space matching their content width.

<span> text <strong> bold

Accessible Media Standards

Always provide descriptive alternate text (the alt attribute) on image assets to assist screen readers and improve search rankings.

External links should use the target="_blank" attribute to open resources in a new browser tab, keeping the dashboard active.

Incorporating CSS Styles

You can apply styles to your HTML code in three distinct ways:

  • Inline styles: Written directly inside the element tag (eg., <p style ="color:blue;">).

  • Internal stylesheet: Written inside custom <style>

  • External stylesheet: Kept inside standolone .css files and loaded using a linked referance (e.g. <link rel="stylesheet" href="index.css">)
